## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/imkaranks/canvas-animations.git
   ```

1. Install required dependencies:

   ```bash
   cd canvas-animations
   npm install
   ```

1. Start your app in development mode:
   ```bash
   npm run dev
   ```
